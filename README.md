# AI群聊 - AI Chat App

一个漂亮的多AI对话界面，使用毛玻璃效果和绿色主题。

## 运行项目

### 前置要求

需要安装 Node.js (建议版本 18+)。

下载地址: https://nodejs.org/

### 安装步骤

1. 打开终端/命令提示符，进入项目目录：

```bash
cd ai-chat-app
```

2. 安装依赖：

```bash
npm install
```

3. 启动开发服务器：

```bash
npm run dev
```

4. 在浏览器中打开 http://localhost:5173 查看效果

## 技术栈

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Motion (Framer Motion)
- Lucide React Icons
