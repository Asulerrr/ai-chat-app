# AI群聊 - 多AI对话应用

一个支持多AI模型同时对话的现代化Web应用，采用毛玻璃设计风格。

## 项目位置

```
C:\Users\95448\ai-chat-app
```

## 技术栈

- **React 18** - 前端框架
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **Tailwind CSS** - 样式框架
- **Motion (Framer Motion)** - 动画库
- **Lucide React** - 图标库

## 运行项目

```bash
cd C:\Users\95448\ai-chat-app
npm install
npm run dev
```

访问 http://localhost:5173 查看应用

## 功能特性

### 1. 主题切换
- **自动检测系统主题** - 首次加载自动匹配Windows深色/浅色模式
- **手动切换** - 点击右上角日/月图标切换主题
- **记住选择** - 手动切换后保存到本地存储，刷新后保持

### 2. 历史会话管理
- **会话列表** - 左侧显示历史会话列表
- **新建会话** - 点击底部消息图标创建新会话
- **切换会话** - 点击列表项切换到该会话
- **展开/收起** - 点击侧边栏按钮收起/展开历史列表
- **右键操作** - 右键会话项显示操作按钮：
  - 📌 置顶/取消置顶
  - ✏️ 修改会话名称
  - 🗑️ 删除会话

### 3. AI模型管理
- **AI列表** - 点击右下角Bot按钮打开AI列表
- **勾选启用** - 勾选AI模型在聊天区域显示
- **最多6个** - 同时最多支持6个AI模型展示
- **添加AI** - 点击"添加AI"输入名称和链接
- **自动识别** - 输入链接后自动识别AI名称
- **删除AI** - 悬停显示删除按钮

### 4. 聊天区域布局
根据激活的AI数量自动调整布局：
- **1个AI** - 全屏展示
- **2个AI** - 两列并排
- **3个AI** - 三列并排
- **4个AI** - 2×2 网格
- **5个AI** - 上3下2 布局
- **6个AI** - 上3下3 布局

### 5. AI切换与交换
- **下拉切换** - 点击聊天列标题的下拉箭头
  - 点击未使用的AI → 替换当前位置
  - 点击正在使用的AI → 交换两者位置
- **展开模式** - 点击展开按钮单独查看一个AI
- **展开按钮** - 仅在有2个及以上AI时显示

### 6. 输入交互
- **发送按钮状态**
  - 无内容：粉红色，点击时抖动并提示"请输入内容"
  - 有内容：绿色，点击后发送
- **新建会话** - 点击消息图标创建新会话

## 设计特点

- **毛玻璃效果** - 使用 backdrop-blur 实现半透明模糊效果
- **绿色主题** - 采用 emerald 色系作为主色调
- **平滑动画** - 使用 Motion 库实现流畅的交互动画
- **响应式设计** - 适配不同屏幕尺寸
- **深色模式** - 完整支持深色主题

## 快捷操作

| 操作 | 说明 |
|------|------|
| 右键会话项 | 显示置顶/修改/删除按钮 |
| 点击消息图标 | 新建会话 |
| 点击Bot图标 | 打开AI列表 |
| 点击下拉箭头 | 切换/交换AI模型 |
| 点击展开按钮 | 单独查看某个AI |
| 点击主题图标 | 切换日/夜间模式 |
